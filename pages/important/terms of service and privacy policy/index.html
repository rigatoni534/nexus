Privacy Policy for Nexus

This Privacy Policy explains what data the Bot collects, why it is collected, how it is used, how long it is retained, and how you can contact the developer. By inviting or using the Bot you acknowledge and agree to the collection and use described below.

Effective date: Nov 29, 2025

Data Collected

We collect only the following Discord-related data necessary for the Bot to function:

  • Server ID and Server Name
  • Channel ID and Channel Name
  • Current User ID and Username (rotated frequently)
  • Last User ID and Username (rotated frequently)
  • Duration of User (how long a user is active in a session)
  • Messages User Can Say (the content the user sends that the Bot needs to process)

Purpose of Collection

The collected data is used solely to provide and maintain the Bot’s features. Specifically, the data is used to route and deliver messages to the correct server and channel, track the current and previous user to support session continuity, enforce duration or rate limits, and validate permitted message content required for the Bot’s operation. The Bot does not use the data for advertising, profiling, or selling to third parties.

Data Retention

Data is retained only as long as necessary to provide the Bot’s functionality. User identifiers are rotated frequently to minimize stored personal data. If the Bot is removed from a server, associated server- and channel-level data will be deleted within a reasonable timeframe unless retention is required by law.

Sharing and Security

The Bot does not share collected data with third parties except when required by law or to comply with Discord’s developer policies. Reasonable technical and organizational measures are used to protect stored data and limit access to the developer and authorized maintenance systems. Absolute security cannot be guaranteed; users should be mindful of information posted in public channels.

User Choices and Rights

To stop data collection for a server, remove the Bot from that server. You may request deletion of stored data by contacting the developer at the address below. The Bot does not collect email addresses or payment information. If you believe data has been collected in error, contact the developer.

Changes to This Policy

This policy may be updated from time to time. The effective date above will be updated when changes occur.
